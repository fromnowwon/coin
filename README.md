## Introduction
- 1위부터 100위까지 암호화폐 실시간 시세 정보를 출력하는 페이지입니다.
- API : [coinpaprika](https://api.coinpaprika.com/)

<br/>

## Development Environment
- Client: Next.js, TypeScript, SCSS
